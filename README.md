**Dashly** is a full-stack productivity dashboard built with **(Postgresql, Express.js, Next.js, Node.js)**. It helps users manage tasks efficiently with a modern, interactive interface.

---

## 🌟 Features

- User authentication (Sign Up / Sign In)
- Add, update, delete tasks
- Categorize tasks for better organization
- Responsive UI with Next.js + Tailwind CSS
- REST API backend with Node.js & Express.js
- Database: Postgresql
- Environment variables managed securely via `.env`

## ⚡ Installation & Setup

### 1. Clone the repository
```bash
git clone https://github.com/Yasiralishah21/Dashly.git
cd Dashly
2. Backend Setup
cd Backend
npm install
cp .env.example .env   # create your own .env based on example
npm start
3. Frontend Setup
cd ../Frontend
npm install
npm start
4. Access the app

Frontend runs on: http://localhost:3000

Backend API runs on: http://localhost:5000


📝 Notes

Ensure your .env file has correct database credentials and API secrets.

.env files are ignored in Git for security.

Optionally include a .env.example to show placeholders for variables.
